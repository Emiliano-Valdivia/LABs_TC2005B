Laboratorio 1

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Los lenguajes se parecen en que son lenguajes de alto nivel orientados a objetos
    • Los lenguajes también tienen una sintaxis bastante parecida, pues ambos se parecen en cierta medida a C, pero fuera de esto, son bastante diferentes
    • JavaScript es un lenguaje interpretado, mientras que Java se compila
    • Los programas de Java se ejecutan en la máquina virtual de Java, mientras que los de JavaScript se ejecutan en el navegador web
  2. ¿Qué métodos tiene el objeto Date?
    • getFullYear(): Este método regresa el año como un número de 4 digitos
    • getMonth(): Este método regresa el mes como un numero del 0 al 11
    • getMinutes(): Este método regresa el minuto de la hora establecida como un número del 0 al 59
    • getMiliseconds(): Este método regresa el milisegundo del segundo establecido como un número del 0 al 999
    • getTime(): Este método regresa los milisegundos transcurridos desde el 1 de enero de 1970
  3. ¿Qué métodos tienen los arreglos?
    • push(): Introduce información a un arreglo
    • unshift(): Introduce información al principio del arreglo
    • pop(): Saca un elemento del final del arreglo
    • find(): Busca un elemento en específico en un arreglo y regresa true o false, dependiendo de si este se encuentra o no
    • findIndex(): Similar a find(), pero regresa el indice del primer elemento que coincida con el solicitado
  4. ¿Cómo se declara una variable con alcance local dentro de una función? Se puede utilizar la palabra var, ya que esta indica que el alcance de la variable que se declare, será el de la función más cercana.
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones? Utilizar variables globales en lugar de locales puede hacer que se generen conflictos entre funciones si es que usan la misma variable para diferentes propósitos. Esto es especialmente común en códigos que involucran a varias personas.
